# Flower Shop Website

A modern and responsive Flower Shop website built using HTML, CSS, Bootstrap, and Font Awesome.

## Project Overview

This project is a frontend flower shop website created as an assignment.  
The design includes a modern e-commerce layout with responsive sections and attractive UI components.

## Features

- Responsive Navbar with Dropdown Menus
- Hero Slider Section
- Product Cards with Ratings & Badges
- Featured Products Section
- Promotional Banner
- Testimonial Slider Design
- Flower Experts Section
- Blog Section with Comment Count
- Newsletter Subscription Section
- Responsive Footer
- Mobile Responsive Design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- Google Fonts
